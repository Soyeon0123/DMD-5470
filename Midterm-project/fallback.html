Fallback Methods

When CSS custom properties are not supported, fallback values can be defined using the `var()` function with a fallback value.

Case1

If the value is not declared

If a CSS custom property is not declared, the fallback value is used instead.

Case2

Use invaluable properties

The fallback value is used if the CSS custom property is not defined.

 padding-bottom: var(--spacing-xs) var(--spacing-sm);

Browser Support

Browser Support for CSS Custom Properties

CSS custom properties are widely supported across modern browsers, including Chrome, Firefox, Safari, and Edge. However, older versions of Internet Explorer do not support CSS custom properties. In this case, you will need to provide fallback values or use alternative methods for styling. Check out for browser support details on Can I Use - CSS Variables.

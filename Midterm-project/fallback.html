Fallback Methods

When using CSS custom properties, you can define a fallback value inside the var() function. The fallback value will be used if the custom property is not available or cannot be resolved.

Use Case 1

Variable is not declared

If a CSS custom property has not been defined anywhere in the stylesheet, the fallback value will be used.

Use Case 2

Variable value is invalid

Fallback values are also used when the custom property's value cannot be applied to the CSS property.

 var( --primary-color, black);

//If --primary-color is not defined, 
the browser will use black instead.//

Browser Support for CSS Custom Properties

CSS custom properties are widely supported across modern browsers, including Chrome, Firefox, Safari, and Microsoft Edge. However, Internet Explorer does not support CSS custom properties. For projects that must support older browsers, developers may need to provide fallback values or alternative styling approaches.

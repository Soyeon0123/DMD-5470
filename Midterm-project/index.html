CSS Custom Properties (CSS Variables)

CSS Custom Properties, also known as CSS Variables, are reusable values defined in a CSS document that can be referenced throughout a stylesheet to simplify styling and improve maintainability. By defining a value once and reusing it throughout a stylesheet, developers can manage design elements more efficiently.

CSS variables can store many types of values such as colors, spacing, layout values, or design tokens used in web interfaces.

How to Use CSS Variables

Step 1

Declare variables in :root

To define a CSS custom property, use the double hyphen (--) prefix before the variable name. Custom properties follow the cascade and inheritance rules of CSS, meaning they can inherit values from parent elements. Variables are commonly declared inside the :root pseudo-class, which represents the root element of the document. Variables defined in :root become global variables and can be accessed from anywhere in the stylesheet.

Step 2

Use the var() function

After defining custom variables, you can reference them using the var() function. The var() function inserts the value of a custom property into a CSS property. It can only be used inside property values, not in property names or selectors.

Step 3

Override variables

CSS variables can be redefined within a specific scope, such as inside a class or element selector. Because CSS variables follow the cascade and inheritance model, redefining a variable locally overrides its value only within that scope. This allows different sections or themes of a website to use different values while keeping the same variable name. 


:root {
--main-color: blue;
}

.card{
--main-color: red;
}

Step 4

Naming conventions

CSS custom property names may include letters, numbers, hyphens, and underscores, but they cannot contain spaces. They are also case-sensitive. Example:


--main-color
--Main-color
These two variables are treated as different properties. Using descriptive names improves readability and makes stylesheets easier to maintain. For example: 

--Primary-color
is easier to understand than directly using: 

#ffffff

Benefits of Using CSS Variables

1. Easy to update and maintain

Values defined once can be reused throughout the stylesheet. Updating a variable automatically updates every place where it is used.

2. Improved readability and semantics

Meaningful variable names make stylesheets easier to understand and maintain. Instead of repeating raw values like hex colors, variables communicate the purpose of the value.

3. Theming

CSS variables make it easy to implement theme switching (such as light and dark modes). Changing a few variable values can update the entire appearance of a site. This page demonstrates theme switching using the buttons below.

[data-theme="brown"]{
    --bg-color:#ece4da;
    --primary-color: #282828;
    --secondary-color: #583c2b;
    --color-text: #28282b;
    --color-text-secondary: #583c2b;
    --color-accent: #282828;
    --color-text-muted: #7a6e62;
    --corner-color: #e2c660;
  }

4. Responsive design

CSS variables can be combined with media queries to adjust values based on screen size or device conditions. This allows styles to adapt dynamically across different screen sizes. 


@media (max-width: 768px) {
  :root {
    --spacing-unit: 12px;
  }
}

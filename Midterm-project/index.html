CSS custom properties

CSS custom properties are entities that you can define your customized design systems including layout, design tokens, colors, and other visual elements for reusing in your web design.

How to Use?

Step 1

Declare Function in :root

Use a prefix of two dashes (--) followed by a name for your custom property. Custom properties can be referenced globally (:root) pseudo-class.

Step 2

Use var()

Use the var() function to reference a CSS custom property value in your styles. The var() fuction is using only for property values not for property names or selectors.

Step 3

Override variables

You can override CSS custom properties in a specific scope (e.g., a class or element) by redeclaring them.


:root {
--main-color: blue;
}

.card{
--main-color: red;
}

Step 4

Naming Convention

Custom properties' names are case sensitive. (e.g., --main-color is different from --Main-Color)

:root {
  --bg-color:#ece4da;
  --primary-color: #282828;
  --secondary-color: #583c2b;
  --color-text: #282828;
  --color-text-secondary: #583c2b;
  --color-accent: #ae1a1b;
  --color-accent-secondary: #0042AF;
  --color-accent-gold: #e2c660;
  --color-accent-green: #283618;
  --color-text-muted: #7a6e62;
}
